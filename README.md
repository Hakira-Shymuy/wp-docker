## How to use

- Download zip
- extract
- change .env values to whatever you need
- cd wp-app ( or rename folder and cd into it )
- docker compose up --build ( to build images first time, download everything, etc)
- docker compose up ( to run without build )

## Dependencies

- Docker
- Docker Compose CLI
